# MFC_study



##### MessageDriven방식
-  GetMessage()
함수를 통해 메시지 큐(Message Queue)에서 메시지(Message)를 가져옴

- OnCreate() 
WM_CREATE 메시지 발생 시 자동으로 호출되는 메시지 핸들러 함수

- public:  CWnd m_wndChild;
솔루션의 헤더의 클래스 내부의 재정의 부분
CWnd는 창 클래스의 기본 기능을 사용하기 위한 클래스
